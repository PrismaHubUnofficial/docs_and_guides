    Internal Tutorials and Guides Repository

    Overview

    This repository is intended to collect tutorials, technical documents, and operational guides for internal members only.

    The goal of this repository is to provide a centralized and structured knowledge base covering procedures, best practices, and practical instructions for using internal tools, robots, and software systems.

    All contents are meant to support: - onboarding of new members - knowledge transfer within the team - safe and correct operation of shared hardware and software resources

    Available Guides

    General Quick Guides

    A collection of general-purpose quick guides is also available.

    These guides cover: - text-to-speech workflow for video voiceovers - subtitle creation in .srt format for voiceover videos - website update and publication workflow with MkDocs

    📄 References:
    - General Quick Guide/QUICK GUIDE — Voiceover for a Video.md - General Quick Guide/QUICK GUIDE — Creating Subtitles.md - General Quick Guide/QUICK GUIDE — How to update the website.md

    Contribution Guidelines

    • All documents should be written in Markdown (.md)
    • Content should be clear, concise, and reproducible
    • Guides must assume internal usage and may reference internal hardware or configurations

    When adding new material: - create a dedicated folder for each platform or system - update this README with the new reference

    Intended Audience

    This repository is intended exclusively for internal team members. External distribution of the material should be evaluated on a case-by-case basis.
