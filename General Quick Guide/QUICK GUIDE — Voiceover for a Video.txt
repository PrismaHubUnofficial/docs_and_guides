QUICK GUIDE — Voiceover for a Video with Text-to-Speech (TTS)

1) Prepare the script (common)
   - Write the narration.
   - Add punctuation to control pauses.
   - If it’s long, split it into short chunks (≈15–60s each).

   Optional: make it “TTS-ready” with an AI (no word changes).
   Prompt:
   Please prepare this script for text-to-speech.
   Rules:
   - Do NOT change, add, or remove any words.
   - Only adjust punctuation, capitalization, spacing, and line breaks.
   - Add line breaks/extra punctuation to improve pauses.
   Output ONLY the cleaned script.

2) Generate the voiceover (choose one)

   A) Hugging Face TTS (recommended: Kokoro)
      - Open https://huggingface.co/models?pipeline_tag=text-to-speech and select a model (recommended https://huggingface.co/spaces/hexgrad/Kokoro-TTS)
      - Paste one chunk, pick a voice, Generate, preview.
      - If it sounds wrong: add punctuation / split the chunk / regenerate.
      - Download the audio (MP3/WAV depending on the tool).

   B) Microsoft Clipchamp (Text-to-Speech)
      - Open a project, import video, put it on the timeline.
      - Go to: Record & create → Text to speech.
      - Choose language + voice (adjust pace/pitch if available).
      - Paste one chunk → Preview → Save/Add to timeline.
      - Export the voiceover as Audio-only (MP3).
