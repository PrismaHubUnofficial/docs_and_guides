## Pre-Operation Checklist

- [ ] Chassis and wheel status inspected
- [ ] Battery and power distribution checked
- [ ] Network/teleop connection verified
- [ ] Braking/safe-stop behavior tested
- [ ] Operating area secured

## Startup Procedure

1. Power on compute and control units.
2. Verify communication with sensors/actuators.
3. Confirm ready state in teleoperation interface.

## Basic Operation

1. Enable manual or autonomous mode.
2. Perform low-speed movement test.
3. Start the mission profile.

## Shutdown Procedure

1. Stop active mission.
2. Set platform in safe idle mode.
3. Power off control and compute units.

## Troubleshooting

- Problem: No motion response
- Check: Motor enable state and safety lock
- Action: Reset safety state and re-enable drive

## Maintenance Notes

- Daily checks: Wheels, connectors, battery
- Weekly checks: Sensor alignment and logs
- Critical consumables: Batteries, fuses
